# thumbnail-analyzer

> **YouTube thumbnail image → CTR score, design analysis, improvement suggestions.** Color contrast, face emotion, text readability, psychological triggers, A/B test recommendations.

[![PyPI](https://img.shields.io/pypi/v/thumbnail-analyzer?style=flat)](https://pypi.org/project/thumbnail-analyzer/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

## Quickstart

```bash
pip install thumbnail-analyzer
python -m thumbnail_analyzer thumbnail.jpg
python -m thumbnail_analyzer https://img.youtube.com/vi/VIDEO_ID/maxresdefault.jpg
```

## What's analyzed

- **CTR score** A–F with benchmark percentile
- **Visual profile** — contrast, face size, emotion, eye contact, background complexity
- **Text analysis** — word count, readability, color contrast, mobile legibility
- **Psychological triggers** — curiosity gap, social proof, urgency, FOMO, humor
- **Ranked improvements** — specific changes with expected CTR lift
- **Better text overlays** — 2–3 alternative copy suggestions
- **A/B test pairs** — specific hypotheses to test
- **Mobile check** — readable at small sizes
